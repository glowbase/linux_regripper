# RegRipper 4.0 for Linux



### Installation
Ensure you have Perl installed prior to running the following.

```bash
perl -MCPAN -e install Parse::Win32Registry
```

### Use
```bash
./rip.pl -r <hive> -a
./rip.pl -r <hive> -p <plugin>
```
